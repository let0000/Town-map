# Town-map
React와 kakao Map API 를 이용한 지도 검색 사이트

## 배포링크
https://let0000.github.io/townMap

## 기능
키워드와 장소를 입력받아 검색하는 기능\
kakao map API 에서 해당하는 데이터를 받아오는 기능\
마커를 클릭하면 해당 아이템으로 포커스가 맞춰지는 기능

## 라이브러리
react-router-dom\
react-kakao-maps-sdk\
@mui/material\
@mui/icons-material\

## 사용 API
kakao Map API

## 상태관리
reducer 를 이용한 상태관리
